**Задача**
Провести эмоциональную оценку последних n твитов пользователя, его друзей и их друзей с помощью нейросети. Собрать данные, обработать и построить график зависимости эмоциональной оценки от количества твитов, разделить пользователей на кластеры, для каждого из которых определить наиболее характерное и точно подходящее слово.

Планируем использовать TensorFlow и RapidMiner для создания и обучения нейросети. Selenium + Python для парсинга твиттера.

Пример эмоционального анализатора фраз: https://www.paralleldots.com/emotion-analysis
